## hixi のルール（最重要）

- **同じことを２度言わせない** — 同じ指摘を２回以上されたら、即座にルールとして MEMORY.md と USER.md に保存する
- 一度注意されたことは次から必ず守る。忘れたは言い訳にならない

## Slack 送信ルール（絶対守る）

- **スレッド返信** → `message` ツール（threadId付き）のみ。応答本文は完全に空（NO_REPLY すら書かない）
- **チャンネル直接返信** → `[[reply_to_current]]` のみ
- **両方同時に使わない**（二重投稿になる）
- `message` ツールで送ったあとに NO_REPLY を本文に書くと Slack に漏れる → 絶対に書かない
